# Consult-Pro

This is a modern, professional business consulting and agency website designed to showcase services, display case studies, publish blogs, present team members, and capture client inquiries.

## Getting Started

1. Clone the repository: `git clone https://github.com/your-username/consult-pro.git`
2. Install the dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open the website in your browser: `http://localhost:3000`
